# 🚀 Advanced Conversion Funnel Simulator

An interactive decision-support tool that simulates how improvements in user conversion rates impact revenue and overall business performance.

---

## 🌐 Live Demo

https://funnel-simulator.streamlit.app/

---

## 💡 Overview

This project goes beyond traditional funnel analysis by enabling **what-if scenario simulation**.  
It allows users to model different conversion strategies and instantly evaluate their impact on revenue and growth.

---

## 🔥 Key Features

- 🎛 **Scenario Builder** – Adjust conversion rates dynamically
- 📊 **Base vs Scenario Comparison** – Visual funnel comparison
- 💰 **Revenue Impact Analysis** – Calculate projected revenue
- 📈 **Performance Lift (%)** – Measure improvement over baseline
- 📥 **Export Functionality** – Download results as CSV
- 🧠 **Smart Insights** – Automated evaluation of strategies

## ⚙️ Setup & Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/funnel-simulator.git
cd funnel-simulator
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
streamlit run app.py
```
